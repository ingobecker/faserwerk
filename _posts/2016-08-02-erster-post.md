# Jekyll rockt!

Eine Grunderneuerung der faserwerk Seite stand schon lange aus. Der Anteil der Besucher mit Mobilgeräten übersteigt schon seit einiger Zeit die der klassischen Besucher die einen Desktopcomputer verwenden. Die alte Faserwerkseite lässt sich nur beschränkt auf mobilen Endgeräten betrachten und so lag es nahe, dass eine Neuentwicklung der Seite erforderlich war. Das Konzept: eine komplett responsive Seite die sowohl für Mobil- als auch für Desktopbenutzer funktioniert. Als Grundlage bietet sich Jekyll an. Damit lassen sich einfach statische Seite mit simpler Blogfunktion umsetzen. Das spätere Überführen eines Jekyllprojektes in ein Railsprojekt ist relativ einfach und hält die Möglichkeit offen, aus der statischen Seite eine aufwändigere dynamische Webanwendung zu machen, falls das irgendwann benötigt wird.
